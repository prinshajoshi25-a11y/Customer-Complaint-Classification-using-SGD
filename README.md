# Customer-Complaint-Classification-using-SGD
A machine learning system that classifies customer complaints into predefined categories using the SGD (Stochastic Gradient Descent) algorithm.
Project Files
app.py — Flask application
complaint_classifier.pkl — Trained ML model
tfidf_vectorizer.pkl — TF-IDF vectorizer
templates/ — HTML frontend
FinalCustomerComplaintClassificationUsingSGD.ipynb — Model training notebook
